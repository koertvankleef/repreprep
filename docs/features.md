# Features

This page captures current product capabilities in user-facing terms.

## Core Experience

- Local-first workout tracking in the browser.
- No account required.
- No cloud sync required for normal use.
- Nearly never needs you to update.

## Workout Logging

- Log workouts by date.
- Track rep-based sets with confirmation flow.
- Track time-based sets with explicit Start and Stop.
- Rest flow with pause/resume and skip controls.

## Offline and Installability

- Installable as a web app (PWA-style experience).
- Core app shell is cached for offline use after first online load.
- Workout logging can continue without internet when the app is installed and cached.
- UI fonts are self-hosted and cached, so typography remains available offline.

## Data Ownership and Trust

- Data is stored locally in browser storage.
- Import and export JSON backups.
- Human-readable export format.

## Accessibility and UX Direction

- Ongoing accessibility and interaction refinements are tracked in `docs/todo.md`.
- Current priorities include timed-flow interaction improvements and state-flow hardening.
